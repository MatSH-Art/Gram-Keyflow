# Summary
This extension adds powerful keyboard navigation and UI customization to Instagram, letting you browse and interact with posts without touching your mouse. With customizable shortcuts, visibility controls, and smooth navigation, this extension provides a seamless way to enhance your Instagram experience.

Features

- Sleek Minimalist UI: Clean, unobtrusive settings popup with segmented controls and smooth transitions for effortless configuration.
- Keyboard Navigation: Navigate through posts using customizable keyboard shortcut.
- Quick Interactions: Like, bookmark, follow, and control media without clicking; all from your keyboard.
- Carousel Control: Navigate through multi-image posts with dedicated shortcuts.
- Audio Toggle: Mute/unmute video posts instantly with a single keypress.
- Profile Access: Jump directly to post authors' profiles with one keystroke.
- UI Visibility Control: Hide Instagram's sidebar recommendations to reduce distractions with a simple toggle.
- Custom Keybinds: Personalize all keyboard shortcuts to match your workflow and preferences.

# Download the Extension:

- Download this repository as a ZIP file.
- Extract the contents to a folder on your computer.
# Enable Developer Mode:

- Open your browser and navigate to chrome://extensions/.
- Enable Developer Mode (toggle in the top-right corner).
# Loading the Extension:

- Click Load unpacked.
- Select the folder containing the extracted extension files.
- Start Using the Extension:

The extension will appear in your browser toolbar.

Open the popup to start Navigating Instagram like an Expert!
